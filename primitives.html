Primitives

The everyday building blocks — buttons, fields, controls, overlays — all wearing Clean & Modern. This is the kit an agent (or you) reaches for so nothing has to be invented from scratch.

How these ship: in a real app these come from Hero UI, re-themed once via heroui/clean-modern.theme.css — so every Hero UI component is born looking like this. This page is the visual reference for what that produces. The signature pieces (Command Dock, Metric Card, Action Grid) live in the Gallery.

What is the Clean & Modern style?
A calm, premium, native-feeling system — monochrome chrome, one green accent, soft elevation, and a single bounce. Everything in this library wears it.
Where do these components come from?
Generic primitives come from Hero UI, re-themed to these tokens. The signature pieces are hand-built. You browse them here.
Can I use it in a new project?
Yes — import the tokens and the Hero UI theme, then compose screens. An agent should rarely need to invent anything.
